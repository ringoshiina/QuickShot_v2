# Landcloud QuickShot

专为 Landcloud 平台定制的截图工具，支持自动提取项目/地块编号并按规范命名保存。

## 安装步骤 (Installation)

1.  **打开扩展程序页面**：
    *   在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。
    *   或者点击浏览器右上角的三个点图标 -> "扩展程序" -> "管理扩展程序"。

2.  **开启开发者模式**：
    *   在页面右上角找到“开发者模式”开关，并将其打开 (Toggle On)。

3.  **加载扩展程序**：
    *   点击左上角的“加载已解压的扩展程序”按钮。
    *   在弹出的文件选择窗口中，找到并选中本项目所在的文件夹 `QuickShot_v1`。
    *   点击“选择文件夹” (Select Folder)。

4.  **确认安装**：
    *   此时你应该能在列表中看到 "Landcloud QuickShot" 扩展程序。
    *   建议点击浏览器右上角的“拼图”图标，将 QuickShot **固定 (Pin)** 到工具栏，方便使用。

5.  **更新扩展**：
    *   如果代码有更新，只需回到 `chrome://extensions/` 页面，找到该扩展，点击右下角的刷新图标 (旋转箭头) 即可。

## 设置面板
- 保存路径始终位于浏览器默认下载目录下，但可在“根目录”输入框中填写多级子目录（使用 `/` 或 `\` 分隔）。
- 面板提供“配置快捷键...”按钮，可直接打开 `chrome://extensions/shortcuts`，若快捷键无效请在该页面重新绑定。
- 所有设置存储在 `chrome.storage.sync` 中，自动在同账号 Chrome 之间同步。
- 工具栏弹窗与完整“选项”页共用同一表单，任意一处保存都会立即生效。

## 自定义
- 默认根目录可在“选项”面板修改，或直接调整 `sw.js` 顶部的 `DEFAULT_SETTINGS.outputRoot`。
- 若页面结构变化，可调整 `scrapeContext` 中的选择器和正则。

## 注意
- 首次调用可能会弹出调试权限授权，允许后即可正常使用。
- 截图输出位于浏览器默认下载目录下的子目录，例如：`F24/F24E81010/F24E81010-09-1.png`。
- 只有默认下载目录及其子目录可写入；如需使用新的根目录，请在设置面板保存后再次截图。
